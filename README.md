# US Political/Emotional SentiMap

SentiMap is a webapp using indico's APIs to analyze Tweets (via Twitter's API) and visualize them on a map using D3. This was created for Lab 2 of OlinJS Spring 2016 by [Brenna Manning](https://github.com/BrennaManning), [Nora Mohamed](https://github.com/nmohamed), and [Jennifer Wei](https://github.com/jenwei).
