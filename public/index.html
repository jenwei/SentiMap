SentiMap


Positive/Negative Sentiment showcases recent tweets pulled from the Twitter Stream API and analyzes it using the indico sentiment analysis API. The result is a series of points plotted on the map above using D3 where the color corresponds to how positive or negative a tweet is and the opacity corresponds to the strength of the sentiment.

Political Sentiment showcases recent tweets pulled from the Twitter Stream API and analyzes it using the indico political sentiment analysis API. The result is a series of points plotted on the map above using D3 where the color corresponds to the highest likelihood score of the four political parties that indico looks scores for (Libertarian, Conservative, Liberal, Green), and the opacity corresponds to the strength of the highest likelihood score.
